# Finance Analyser

Local personal finance dashboard for consolidating statements from four financial institutions (Commerzbank, Wise, Trade Republic, TF Bank) into a unified view of income, expenses, category breakdown, and net worth over time.

Backend is ASP.NET Core (.NET 8); frontend is Vite+React. The two run together via Docker Compose for normal use.

## Running the full stack (Docker Compose)

```bash
docker compose up -d --build
```

- `frontend` — built and served by nginx on http://localhost:8080, proxies `/api/` to `api:3001`.
- `api` — ASP.NET Core API on port 3001 (internal to the compose network).
- Bind-mounts `./data` (SQLite DB) and `./docs` (source PDFs/CSVs for import) into the `api` container.
